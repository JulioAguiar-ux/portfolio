## Tecnologias Utilizadas

Este portfólio e os projetos nele apresentados utilizam uma variedade de tecnologias modernas, incluindo:

*   **HTML5:** Para a estruturação semântica do conteúdo web.
*   **CSS3 (com CSS Grid):** Para estilização e layouts responsivos e eficientes.
*   **JavaScript:** Para interatividade e dinamismo nas interfaces.
*   **React:** Para o desenvolvimento de interfaces de usuário reativas e componentes reutilizáveis.
*   **NestJS:** Para a construção de APIs robustas e escaláveis no backend.
*   **JWT (JSON Web Token):** Para autenticação segura e controle de acesso.

## Como Visualizar e Configurar

1.  **Organize suas imagens:** Salve sua foto de perfil como `assets/img/foto-ai.png` e as imagens dos seus projetos conforme a estrutura de pastas acima.
2.  **Abra no navegador:** Para visualizar o portfólio localmente, basta abrir o arquivo `index.html` em seu navegador preferido (ou utilize o Live Server no VS Code para desenvolvimento).

## Hospedagem no GitHub Pages

Para hospedar seu portfólio gratuitamente no GitHub Pages, siga estes passos:

1.  **Crie um repositório:** Crie um novo repositório público no GitHub (ex: `portfolio`).
2.  **Faça o upload:** Faça o upload de toda a estrutura da pasta `portfolio/` para este repositório, utilizando `git push` ou o próprio site do GitHub.
3.  **Configure o GitHub Pages:** Vá para **Settings** → **Pages** no seu repositório, selecione a branch `main` (ou `master`) e a pasta `/(root)` para o deploy.
4.  **Acesse seu portfólio:** Seu portfólio estará disponível em um URL similar a: `https://JulioAguiar-ux.github.io/portfolio/`.
